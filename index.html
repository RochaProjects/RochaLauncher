Características

Ferramenta de gerenciamento poderoso

Descubra os principais recursos que fazem do Rocha Launcher a melhor escolha para gerenciar o Minecraft Bedrock.

Tela Inicial

Painel principal com detalhes globais

Area principal do launcher com status da instancia ativa, informacoes de versao e atalhos de acesso rapido.

  • Resumo visual da versao ativa
  • Fluxo rapido para iniciar o jogo
  • Atalhos para funcoes principais
Selecionar Versao

Troca de versoes locais e instaladas

Fluxo para selecionar rapidamente qual pacote/versao sera iniciado, mantendo organizacao entre opcoes locais e pacote oficial.

  • Selecao por instancia
  • Compatibilidade por ABI
  • Controle de inicializacao
Gerenciador de Arquivos

Abas para mundos, packs e edicao de options

Interface focada em manutencao de conteudo, com categorias separadas para organizar e operar os arquivos do jogo.

  • Listagem por categoria
  • Acoes de exportar e remover
  • Editor de options integrado
Gerenciar Contas

Controle de perfis e dados de conta

Tela dedicada para visualizar, trocar e administrar contas usadas no launcher com interface clara e focada.

  • Perfis organizados
  • Avatar e identificacao
  • Gestao centralizada
Toolbox

Ferramentas de sobreposicao durante o jogo

Conjunto de utilitarios de overlay para acesso rapido em jogo, com foco em usabilidade e configuracao simplificada.

  • Acesso in-game
  • Acoes rapidas
  • Base para recursos futuros
